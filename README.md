# Contreras Bros Moving Services

Spec concept build for Contreras Bros Moving Services.

This is a conversion-first, mobile-first static site concept for a Wood River Valley moving company.

## Files
- `index.html`
- `styles.css`
- `script.js`

## Preview locally
Open `index.html` directly, or run a local server from this folder:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Notes
- Built as a speculative concept, not an official business website.
- Copy uses public business details gathered from Yelp, Instagram, and public directory listings.
- Public logo/photos were intentionally not embedded; this version stays typography-led until licensed assets or business approval are available.
